Matriz 5x5 en Java
 
Descripción
 
Programa en Java desarrollado como tarea escolar, diseñado para gestionar una matriz de dimensiones 5x5 con operaciones básicas.
 
Contenido
 
-  Matriz5x5.java : Código fuente con la lógica del programa.

- Capturas de pantalla: Evidencias del funcionamiento del código.
 
Cómo ejecutar
 
1. Tener instalado el JDK.

2. En la terminal:
javac Matriz5x5.java
java Matriz5x5
 
Funcionalidades principales
 
- Crear una matriz de 5x5 vacía

- Llenarla automáticamente con números enteros aleatorios

- Mostrar la matriz en pantalla con formato ordenado

- Calcular y mostrar la suma de todos los elementos de la matriz

- Identificar y mostrar el valor máximo y mínimo dentro de la matriz
